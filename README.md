# Projekt PWAI – Analiza danych pogodowych

Projekt dotyczący analizy danych pogodowych, realizowany w ramach przedmiotu **Podstawowy Warsztat AI**.

## Struktura repozytorium

Repozytorium zawiera następujące elementy:

- **data/**  
  Folder zawierający pliki z danymi niezbędnymi do realizacji projektu.

- **src/**  
  Folder zawierający plik *Jupyter Notebook*, w którym znajduje się kod odpowiedzialny za:
  - przygotowanie danych,
  - analizę oraz pracę z danymi pogodowymi.

- **demo/**  
  Folder zawierający plik **PDF** z raportem dotyczącym projektu.

- **requirements.txt**  
  Plik zawierający listę bibliotek wraz z ich wersjami, które zostały użyte w projekcie.
